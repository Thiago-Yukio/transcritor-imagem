# 🖼️ Transcritor de Imagens:

Aplicação desenvolvida em **Python**, utilizando **Streamlit** e **EasyOCR**, para extrair textos de imagens por meio de OCR (*Optical Character Recognition*).

---

## Materiais disponíveis:

* `requirements.txt`
* `runtime.txt`
* `transcritor.py`
* **Projeto em funcionamento:** https://transcritor-imagem.streamlit.app/

---

## Funcionalidades:

* Upload de capturas de tela (*Print Screen*);
* Colagem de imagens da área de transferência;
* Extração automática de texto (OCR);
* Exibição do texto reconhecido.

---

## Tecnologias utilizadas:

* Python 3.11
* Streamlit
* EasyOCR
* NumPy
* PyTorch
